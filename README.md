# DreamPage

Aqui é o portal para as maravilhas desse mundo, um baú com todos os meus tesouros escondidos.

## 💫 Começando

Neste site teremos 4 páginas principais:

* **Filmes**
* **Series**
* **Livros**
* **Jogos**

onde cada uma ira conter páginas tituladas 
semelhantemente a *"Visto"*, *"Não visto"* e *"Em processo"*

Na página inicial do site teremos o campo para preencher com a senha dando acesso as demais páginas

Terá uma página extra que aparecerá após a senha ser colocada que se chamará 

* **O Sentimento:** Apenas para me ajudar do necessário
